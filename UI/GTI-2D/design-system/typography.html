排版规则

✓ 推荐做法

  • • 中英文之间保持适当间距
  • • 数字使用等宽字体显示
  • • 英文专有名词加粗或变色强调
  • • 行高设置为 1.5-1.6 倍

✗ 避免做法

  • • 中英文紧贴，缺乏呼吸感
  • • 数字和中文使用不同字体
  • • 过多颜色变化干扰阅读
  • • 行高过小，文字拥挤

CSS 代码示例

Tailwind CSS 类名

/* 标题样式 */
.game-title {
  @apply text-4xl font-military font-bold
         text-cyber-400 tracking-tight;
}

/* 正文样式 */
.game-body {
  @apply text-base font-military
         text-tactical-200 leading-relaxed;
}

/* 数值显示 */
.game-number {
  @apply font-mono font-bold text-xl
         text-cyber-400 tabular-nums;
}

/* 状态文本 */
.game-status {
  @apply text-sm font-military font-semibold
         uppercase tracking-widest;
}

CSS 自定义属性

:root {
  /* 字体家族 */
  --font-military: 'Inter', system-ui, sans-serif;
  --font-mono: 'JetBrains Mono', monospace;

  /* 字体大小 */
  --text-xs: 0.75rem;
  --text-sm: 0.875rem;
  --text-base: 1rem;
  --text-lg: 1.125rem;
  --text-xl: 1.25rem;

  /* 移动端大小 */
  --text-mobile-lg: 1rem;
  --text-mobile-xl: 1.125rem;

  /* 行高 */
  --leading-tight: 1.25;
  --leading-normal: 1.5;
  --leading-relaxed: 1.625;
}
